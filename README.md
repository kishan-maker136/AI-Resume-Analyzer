# AI Resume Analyzer

## Project Overview
AI Resume Analyzer is a web-based application that analyzes resumes and
matches them with job descriptions using Artificial Intelligence.
It helps recruiters shortlist candidates and provides feedback to job seekers.

## Features
- Resume parsing (PDF, DOCX)
- Skill extraction using NLP
- Job description matching
- Skill gap analysis
- Simple and interactive UI

## System Architecture
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- AI Engine: NLP + Machine Learning
- Output: JSON-based analysis

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Scikit-learn
- NLP Libraries
